# eanglish-words

Приложение для изучения английских слов. Позволяет добавлять, просматривать и повторять слова с переводом. Состоит из серверной части на Node.js и клиентского интерфейса (HTML/CSS/JS).

## Технологии

**Бэкенд:**
- Node.js
- Express.js
- PosgreSQL 18 (файл `database/`)
- dotenv

**Фронтенд:**
- HTML5, CSS3
- Нативный JavaScript
- Сборка статики из папки `public/`

## Быстрый старт

### Предварительные требования
- Node.js >= 16.x
- npm >= 8.x

### Установка и запуск

```bash
# 1. Клонируйте репозиторий
git clone https://github.com/zaturaeva/eanglish-words.git
cd eanglish-words

# 2. Установите зависимости
npm install

# 3. Настройте переменные окружения
# Создайте файл .env (пример структуры внутри проекта)
# При необходимости укажите порт и путь к базе данных

# 4. Запустите сервер
# Сервер запустится на http://localhost:3000 (или том порту, который указан в .env)
node server.js
```

## API
- GET /api/words — получить список всех слов
- POST /api/words — добавить новое слово { original: "cat", translation: "кошка" }
- PUT /api/words/:id — обновить слово
- DELETE /api/words/:id — удалить слово

## Использование
- Откройте http://localhost:3000 в браузере
- Добавляйте английские слова и их переводы через интерфейс
- Повторяйте слова в режиме флеш-карточек (flashcards)
- Все данные сохраняются локально в PosgreSQL
